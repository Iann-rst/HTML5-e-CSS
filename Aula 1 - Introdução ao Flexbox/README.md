## Objetivos da aula

1. Conhecer a estrutura básica;
2. Entender a diferença entre Flex Container e Flex Item;
3. Conhecer inicialmente algumas propriedades;


## Propriedades do Flex Container
* display;
* flex-direction;
* flex-wrap;
* flex-flow;
* justify-content;
* align-items;
* align-content;

## Propriedades relacionadas ao Flex Item
* flex-grow;
* flex-basis;
* flex-shrink;
* flex;
* order;
* align-self;
